# template-cli

命令行工具创建模版框架

模版框架有 nest-demo、vue2、vue3、vite+react，开箱即用
ssr 模版框架开发中。。。

### 安装

`npm install template-cli-zl -g` 



### 下载模板

1. 在项目目录下，执行命令
   `template-cli create`

2. 根据命令交互输入项目名称（默认是demo）

3. 选择模版

4. `cd demo`（或输入的项目名）

   

模板是根据使用过或归纳比较优雅的常用技术栈，具备比较通用的、基本的业务开发配置
如有不足，请自行增减（^_^）
